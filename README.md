# Orca Blank Outdent

虎鲸笔记插件：空白块退缩进。

这个插件会调整虎鲸笔记里空白大纲块的 Enter 行为：当当前块是空白内容时，按 Enter 会让这个块向外退出一层缩进；当前块有文字时，仍然使用虎鲸笔记原本的 Enter 换行逻辑。

## 功能说明

- 空白的文本、列表、任务或标题块按 Enter：当前块退出一层缩进。
- 非空块按 Enter：保持虎鲸笔记默认行为，创建同层新块或拆分块。
- 有文字选区时按 Enter：插件不接管，继续走虎鲸笔记默认行为。
- 顶层空白块按 Enter：插件不再强制退缩进，交给虎鲸笔记默认处理。
- 聚焦视图中不额外判断根层级，由虎鲸笔记自身决定是否允许退缩进。

## 使用方式

1. 将整个 `orca-note-plugin-blank-outdent` 文件夹放到虎鲸笔记的 `plugins` 文件夹里。
2. 重启虎鲸笔记。
3. 在虎鲸笔记的插件设置中启用 `Blank Outdent`。

## 开发与构建

```bash
pnpm build
```

构建后会生成 `dist/index.js`。为了减少虎鲸笔记开发模式下的缓存干扰，插件会把运行代码打包成单个文件。

## 代码学习入口

- `src/index.ts`：插件主入口，负责加载、卸载、监听 Enter，并调用退缩进命令。
- `src/decisions.ts`：判断当前 Enter 是否应该被插件接管。
- `src/dom.ts`：从虎鲸笔记界面里找到当前块和文本内容。
- `src/orca.d.ts`：虎鲸插件 API 类型文件，已更新为新版，适合作为本地 API 字典。
- `docs/annotated-index.js`：学习用的 JS 阅读版，带更白话的中文注释，不参与构建。
- `docs/development-notes.md`：本插件开发过程回顾和后续插件开发参考。

## 手动测试

- 在非空块按 Enter，确认仍然是虎鲸笔记默认换行。
- 在缩进下的空白块按 Enter，确认该块向外退一层。
- 连续按 Enter，确认空白块逐层退出缩进。
- 到顶层后继续按 Enter，确认不再发生重复退缩进。
- 选中文字后按 Enter，确认插件不接管默认行为。
